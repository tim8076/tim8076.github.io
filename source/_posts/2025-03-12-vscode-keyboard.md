## 文案快速插入html標籤

假設有一組文案要加入html標籤

```
部署專案，確認上線環境配置無誤
製作專案簡報 (PPT)，準備展示
添加次要動畫效果（可選）
```

1. 開啟 vscode file => Preference => Keyboard shortcut
2. 輸入 Wrap with Abbreviation 及可設定快捷鍵
3. 將文案全選，使用設定的快捷鍵開啟輸入框
4. 輸入要加入的 html 標籤

範例: ul>li*
上面的星號表示每行文字要個別放入 li 當中

## 尋找檔案

ctrl + p  快速開啟搜尋檔案框

輸入檔案名稱 Home:16
:16 代表到該檔案的16行

## 關閉目前分頁

ctrl + w

## 同時選取多行
全選後，ctrl + shift + l

## 選取相同單字

ctrl + d
