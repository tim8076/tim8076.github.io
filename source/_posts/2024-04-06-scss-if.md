## if 用法

在 scss 中，可以用 @if 進行判斷，如下。

``` scss
@if(10 > 5) {
  .test-if {
    color: black;
  }
} @else {
  .test-if {
    color: white;
  }
}

// 以上會產上 color: black; 樣式
```

## and 用法

當想判斷 兩個值是否同時符合條件時，可以用 and

``` scss
@if(1 == 1 and 5 != 1) {
  .test-if {
    color: black;
  }
} @else {
  .test-if {
    color: white;
  }
}
// 以上會產上 color: black; 樣式
```
