## v-text

vue 除了使用 `{{ }}` 作為模板外，也可以使用指令處理模板

以下寫法相同:

``` html
<p>{{ text }}</p>
<p v-text="text"></p>
```

不同的是如果在 p 標籤內加入文字，v-text 指令渲染的會無視標籤內原本內容，而 `{{ }}` 則會顯示  hello world

``` html
<p>{{ text }} world</p> // hello world
<p v-text="text">world</p> // hello
```

## v-html

v-html 可以用來插入 html 結構

``` html
<p v-html="text">123</p>
<script>
  // vue3 寫法
  const { createApp } = Vue;
  const vm = createApp({
    data() {
      return {
        text: '<h1>我是標題</h1>'
      }
    },
  });
  vm.mount('#app');
</script>
```
