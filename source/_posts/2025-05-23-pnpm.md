## 介紹
pnpm 是一個 Node.js 套件管理工具，和 npm 或 yarn 類似，但使用「硬碟節省機制（hard link）」來更有效率地管理 node_modules，速度快且佔用空間小。以下是常用 pnpm 的指令與用法說明：

## 下載 PNPM

全域安裝 PNPM
```
npm install -g pnpm
```

## 安裝套件

```
pnpm install <套件名稱>
```

## 執行指令

```
npm run <指令名稱> 
```

可以和 npm 一樣用 npm run dev 等指令運行專案
