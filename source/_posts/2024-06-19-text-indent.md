## 基本設定

text-indent 是 CSS 屬性之一，用於設定文字的首行縮排，設定的值以數值為主，也可以用百分比或關鍵字。

可以設定的值包括: px、em、rem、cm 、% 等

``` css

// 首行縮排兩個字大小
p {
  text-indent: 2em;
}
```
