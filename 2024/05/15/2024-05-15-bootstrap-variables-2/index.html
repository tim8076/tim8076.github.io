Bootstrap (2) SCSS 客製化-變數更改

發表於 2024-05-15 更新於 2024-08-13 分類於 Bootstrap
更改SCSS以客製化 Bootstrap

Variable defaults

在 bootstrap 裡的每個 variable 都有加上 !default 標籤，讓我們在自己的 scss 檔案裡能覆蓋這些樣式

網站主色更改

在 bootstrap 資料夾裡 可以找到 __variable.scss 檔案，在裏頭可以修改變數來客製化

__variable.scss 檔案從上到下的順序為 色系 > Options(預設樣式)> Spacing(間隔)> Body > link (連結) > breakpoints(斷點 ˇ)

在 __variable.scss 檔案 找到$theme-colors 來更改顏色

要新增色系直接加在 $theme-colors 裡面即可

options 預設樣式更改

在變數的 Options 區域可以更改全站設定

Spacing 更改間距

在變數的 spacer 區域可以更改距離設定

更改後跟距離有關係的 class 都會吃到效果，如 p-3 、mb-2 等

body 更改全站樣式

body 區域可以更改全站預設樣式

文字大小設定

文字的通用樣式預設無法響應式變化

需要更改 responsive 屬性，修改方式可參考 Bootstrap - Utilities API

然後在 關掉 RFS 功能

清除文字標籤下方空間

  • <h1>~<h6> 標籤：$headings-margin-bottom
  • <p>：$paragraph-margin-bottom
  • 字重：$font-weight-base
  • 行高：$line-height-base

link 更改連結樣式

修改元件樣式

Button

  • 字重：$btn-font-weight
  • 間距：$btn-padding-y、$btn-padding-x
  • 圓角：$btn-border-radius
  • disabled
    • $btn-disabled-opacity

表單

  • input:
    • padding: $input-btn-padding-y、$input-btn-padding-x
    • 字重：$input-font-weight
    • 提示字顏色：$input-placeholder-color
    • 背景色：$input-bg
    • 圓角：$input-border-radius
