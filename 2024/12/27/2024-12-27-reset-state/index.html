發表於 2024-12-27 分類於 React
介紹 React state 保存與重設的方法

狀態(state)與渲染樹中的位置相關

當你給元件添加狀態時，你可能會以為這個狀態是「存在於元件內部」的。但事實上，狀態實際上是由 React 持有的。React 會根據元件在渲染樹中的位置，將它所持有的每一個狀態與對應的元件相關聯。
